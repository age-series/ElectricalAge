# Metric Prefixes In Readouts

Electrical Age tooltips and readouts use metric prefixes for many values. The examples often write the full base-unit value first, then include the likely displayed form.

## Common Prefixes

| Prefix | Meaning | Multiplier | Example |
| --- | --- | --- | --- |
| `µ` micro | one millionth | `0.000001` | `50 µA = 0.000050 A` |
| `m` milli | one thousandth | `0.001` | `5 mA = 0.005 A` |
| no prefix | base unit | `1` | `3.0 V = 3.0 V` |
| `k` kilo | one thousand | `1000` | `1 kV = 1000 V` |
| `M` mega | one million | `1000000` | `2 MW = 2000000 W` |

## Reading Values

- `5 mA` means `5 milliamps`, or `0.005 A`.
- `150 mV` means `150 millivolts`, or `0.150 V`.
- `250 mF` means `250 millifarads`, or `0.250 F`.
- `100 mF` means `100 millifarads`, or `0.100 F`.

The prefix is attached to the unit. For example, `mA` means milliamps, while `mV` means millivolts.
